# API Gateway

forward POST requests from localhost:8070/auth/* to localhost:3002/auth/*

```POST localhost:3002/auth/login``` 	

```POST localhost:3002/auth/register```

```GET localhost:3001/products```	

```GET localhost:3001/products/id```
